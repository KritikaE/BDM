## WEEK 1 & 2 — Microeconomics, Utility & Consumer Behaviour

### Marginal Utility
```text
MU = ΔTU / ΔQ
   = Change in Total Utility / Change in Quantity
```

### Cardinal Utility Equilibrium (Consumer Optimum)
```text
MUx / Px = MUy / Py

i.e., Marginal Utility per rupee spent must be equal across all goods.
```

### Budget Constraint
```text
Px · Qx + Py · Qy = I
Where I = Income, P = Price, Q = Quantity
```

### Marginal Rate of Substitution (MRS) — Ordinal/Indifference Curve
```text
MRS = ΔY / ΔX  (slope of indifference curve, negative)
    = MUx / MUy
```

### Law of Diminishing Marginal Utility
```text
As Q ↑ → MU ↓
When MU = 0 → TU is maximum
When MU < 0 → TU decreases
```

## WEEK 2 — Demand, Supply & Elasticity

### Price Elasticity of Demand (PED)
```text
PED = |% Change in Quantity Demanded / % Change in Price|
    = |(ΔQ/Q) / (ΔP/P)|
    = |(ΔQ/ΔP) × (P/Q)|
```

| Value | Interpretation |
|-------|----------------|
| PED > 1 | Elastic |
| PED < 1 | Inelastic |
| PED = 1 | Unit Elastic |
| PED = 0 | Perfectly Inelastic (vertical curve) |
| PED = ∞ | Perfectly Elastic (horizontal curve) |

### Income Elasticity of Demand (YED)
```text
YED = % Change in Quantity Demanded / % Change in Income
    = (ΔQ/Q) / (ΔY/Y)
```

| Value | Good Type |
|-------|-----------|
| YED > 0 | Normal Good |
| 0 < YED < 1 | Necessity |
| YED > 1 | Luxury |
| YED < 0 | Inferior Good |

### Cross-Price Elasticity of Demand (XED)
```text
XED = % Change in Qty Demanded of Good A / % Change in Price of Good B
```

| Value | Relationship |
|-------|--------------|
| XED > 0 | Substitutes |
| XED < 0 | Complements |
| XED = 0 | Unrelated |

### Price Elasticity of Supply (PES)
```text
PES = % Change in Quantity Supplied / % Change in Price
```

## WEEK 2 — Production & Cost

### Marginal Product (MP) and Average Product (AP)
```text
MP = ΔQ / ΔL   (Change in output per unit change in labour)
AP = Q / L      (Total output per unit of input)
```
- If MP > AP → AP is rising
- If MP < AP → AP is falling
- If MP = AP → AP is at maximum

### Cobb-Douglas Production Function
```text
Q = a · L^b · K^c

Where:
  Q = Output
  L = Labour
  K = Capital
  a = Total factor productivity

Returns to Scale:
  b + c > 1 → Increasing Returns to Scale (IRTS)
  b + c = 1 → Constant Returns to Scale (CRTS)
  b + c < 1 → Decreasing Returns to Scale (DRTS)
```

### Total, Average & Marginal Cost
```text
TC = TFC + TVC
AC = TC / Q
MC = ΔTC / ΔQ
AVC = TVC / Q
AFC = TFC / Q
```

### Profit Maximisation Condition
```text
MR = MC   →  Profit is maximised
ΔProfit = MR − MC
```

### Total, Average & Marginal Revenue (Competitive Market)
```text
TR = P × Q
AR = TR / Q = P  (in competitive market, AR = P always)
MR = ΔTR / ΔQ = P  (in competitive market, MR = P always)
```

## WEEK 3 — Pricing & Financial Ratios

### Markup / Cost-Plus Pricing
```text
Markup % = (Profit / Cost) × 100
Selling Price = Cost × (1 + Markup%)
```

### Target Pricing (Break-even)
```text
Break-even Quantity = Fixed Costs / (Selling Price − Variable Cost per unit)
```

### Contribution Pricing
```text
Contribution = Selling Price − Variable (Direct) Cost
```

### Marginal Cost Pricing
```text
MC = ΔTotal Cost / ΔOutput
```

---

### LIQUIDITY RATIOS

#### Current Ratio
```text
Current Ratio = Current Assets / Current Liabilities
Ideal: 1.5:1
```

#### Acid Test / Quick Ratio
```text
Quick Ratio = (Current Assets − Stock) / Current Liabilities
Ideal: 1:1
```

---

### PROFITABILITY RATIOS

#### Gross Profit Margin
```text
GPM = (Gross Profit / Turnover) × 100
Gross Profit = Revenue − Cost of Goods Sold (COGS)
```

#### Net Profit Margin
```text
NPM = (Net Profit / Turnover) × 100
```

#### Return on Capital Employed (ROCE)
```text
ROCE = (Profit / Capital Employed) × 100
```

#### Gross Margin Percentage (Manufacturing context)
```text
GM% = ((Revenue − COGS) / Revenue) × 100
```

---

### INVESTMENT / SHAREHOLDER RATIOS

#### Earnings Per Share (EPS)
```text
EPS = Profit After Tax / Number of Shares
```

#### Price-Earnings Ratio (P/E)
```text
P/E = Market Price / EPS
```

#### EV/EBITDA
```text
EV/EBITDA = Enterprise Value / EBITDA
```

#### Dividend Yield
```text
Dividend Yield = (Ordinary Share Dividend / Market Price) × 100
```

---

### EFFICIENCY RATIOS

#### Asset Turnover Ratio
```text
Asset Turnover = Sales Turnover / Assets Employed

ROCE = Asset Turnover × Net Profit Margin
```

#### Stock Turnover Ratio
```text
Stock Turnover = Cost of Goods Sold / Average Stock
(expressed as times per year)
```

#### Debtor Days
```text
Debtor Days = (Debtors / Sales Turnover) × 365
(Shorter = better)
```

---

### GEARING RATIO
```text
Gearing = (Long Term Loans / Capital Employed) × 100
(Higher = more exposed to interest rate risk)
```

## WEEK 4 — Industries & Market Structure

### Concentration Ratio (CR_n)
```text
CR_n = Sum of market shares of top n firms
(Usually n = 4)
Higher CR → More monopolistic
Lower CR → More competitive
```

### Herfindahl-Hirschman Index (HHI)
```text
HHI = Σ (market share of each firm)²
Range: 0 to 10,000

HHI < 1000        → Low concentration (competitive)
1000 ≤ HHI ≤ 1800 → Medium concentration
HHI > 1800        → High concentration (monopolistic)
```

## WEEK 5 & 6 — E-Commerce, Pareto, Inventory

### Pareto Principle (80-20 Rule)
```text
~80% of outcomes come from ~20% of causes
e.g., 80% of revenue from top 20% of SKUs
```

### Cumulative Contribution %
```text
Cumulative % = (Cumulative Sum / Total Sum) × 100
```

### Inventory Ledger
```text
Closing Stock = Opening Stock − Sales + Incoming Stock (Transfers/Purchases)
```

### Average Days of Inventory (DOI / DOH)
```text
DOI = Average Stock / Average Daily Sales
    = (Opening Stock + Closing Stock) / 2  ÷  Average Daily Sales
```

### Days of Sales Available
```text
Days of Sales = Current Stock / Average Daily Sales
```

### Safety Stock
```text
Safety Stock = Maximum Demand − Average Demand
             = (Max daily demand − Avg daily demand) × Lead Time
```

### Reorder Point (ROP)
```text
ROP = Lead Time Demand + Safety Stock
    = (Average Daily Demand × Lead Time) + Safety Stock
```

### Economic Order Quantity (EOQ)
```text
EOQ = √(2 × D × S / H)

Where:
  D = Annual Demand (units)
  S = Ordering Cost per order
  H = Holding Cost per unit per year
```

### Working Capital
```text
Working Capital = Current Assets − Current Liabilities
```

## WEEK 7 — Manufacturing, Revenue & Production Planning

### Revenue
```text
Revenue = Quantity Sold × Selling Price
```

### Closing Inventory (Production Planning)
```text
Closing Inventory = Opening Inventory − Sales + Production
```

### Blank/Gear/Assembly Relationship
```text
Gear Assembly requires specific Gears
Each Gear is produced from a specific Blank
Total Blanks Required = Σ (GAs produced × Blanks per GA)
```

### Number of Sales Agents Required
```text
Agents = ceil(Max Monthly Sales / Capacity per Agent)
Agent Cost = Number of Agents × Monthly Salary
```

## WEEK 8 — OEE, Scrap, Margins, Safety Stock

### Overall Equipment Effectiveness (OEE)
```text
OEE = Availability × Performance × Quality

Availability (A)  = Run Time / Planned Production Hours
                  = (Planned Shifts − Breakdown Shifts) / Planned Shifts

Performance (P)   = Actual Machine Speed / Design Machine Speed
                  = Actual Output / Planned Output

Quality (Q)       = Number of Good Products / Total Products Made
                  = Broaching Output / Hobbing Output (assuming scrap only in hobbing)

OEE = A × P × Q
```

### Scrap Calculation
```text
Scrap = Hobbing Output − Broaching Input
Scrap % = (Scrap / Hobbing Output) × 100
```

### Material Cost of Scrap
```text
Scrap Material Cost = Number of Scrapped Units × Cost of Blank per Unit
```

### Unit Level Profitability / Gross Margin
```text
Cost of GA = Direct Material Cost + Direct Labour Cost + Production Overhead

Gross Margin = Average Sales Price − Cost of GA

Gross Margin % = ((Sales Price − COGS) / Sales Price) × 100
               = (Gross Margin / Revenue) × 100
```
> Note: G&A Overhead is excluded from COGS for Gross Margin; it's deducted later to arrive at Net Margin.

### Net Margin
```text
Net Margin = Gross Margin − G&A Overhead
Net Margin % = (Net Profit / Revenue) × 100
```

### Reorder Point & Safety Stock (Manufacturing)
```text
Safety Stock = Maximum Demand per period − Average Demand per period
ROP = (Average Daily Demand × Lead Time in days) + Safety Stock
```

## WEEK 9 — HR, Recruitment & Normalization

### Data Normalization (Range / Max Scaling)
```text
Normalized Value = x / x_max
(when x_min is assumed to be 0)

Full Min-Max Scaling:
Normalized = (x − x_min) / (x_max − x_min)
```

### Z-Score Normalization
```text
Z = (x − μ) / σ
Where μ = mean, σ = standard deviation
```

### Candidate Scoring (Internal Sourcing)
```text
Final Score = Σ (Normalized Positive Attributes) − Σ (Normalized Negative Attributes)
e.g., Score = Norm(Skills) + Norm(Projects) + Norm(Good Appraisals) − Norm(Days to Availability)
```

### Attrition Rate
```text
Attrition Rate = (Number of Employees who Left / Average Number of Employees) × 100
```

### Channel Effectiveness Score
```text
Channel Score = Norm(Application Volume) + Norm(Shortlist %) + Norm(Avg Applicant Rating)
```

## WEEK 10 — Finance & FinTech

### RFM Analysis
```text
Recency  = Days since last purchase (lower = better)
Frequency = Number of purchases in period
Monetary  = Total spend in period

RFM Score = Weighted sum of normalized R, F, M scores
```

### BNPL Revenue & Loss Estimation
```text
Revenue = Approved Loan Amount × Commission Rate (e.g., 3.5%)
Loss    = Defaulted Loan Amount × 100%  (full loss)
Net     = Σ Revenue − Σ Loss
```

### Transaction Drop Rate
```text
Drop Rate = (Users who abandoned cart / Users who initiated checkout) × 100
```

## WEEK 11 — A/B Testing & Credit Risk

### A/B Test — Statistical Significance (Z-test for proportions)
```text
Z = (p1 − p2) / √[ p̄(1 − p̄)(1/n1 + 1/n2) ]

Where:
  p1, p2 = conversion rates of groups A and B
  p̄ = pooled proportion = (x1 + x2) / (n1 + n2)
  n1, n2 = sample sizes
```

### Student's t-test (means comparison)
```text
t = (X̄1 − X̄2) / √(s1²/n1 + s2²/n2)
```

### Credit Risk — Expected Loss
```text
Expected Loss (EL) = PD × LGD × EAD

Where:
  PD  = Probability of Default
  LGD = Loss Given Default (% of loan not recovered)
  EAD = Exposure at Default (loan amount)
```

### CIBIL / Credit Score-based Approval Logic
```text
If Credit Score ≥ Threshold → Approve (up to approved limit)
If Credit Score < Threshold → Decline (with reason: Credit Risk)
```

## GENERAL / CROSS-WEEK FORMULAS

### Percentage Change (Month-on-Month / Year-on-Year)
```text
% Change = ((New Value − Old Value) / Old Value) × 100
```

### CAGR (Compound Annual Growth Rate)
```text
CAGR = (Ending Value / Beginning Value)^(1/n) − 1
Where n = number of years
```

### Market Share
```text
Market Share (%) = (Firm's Revenue or Output / Total Market Revenue or Output) × 100
```

### Break-Even Point
```text
BEP (units) = Fixed Costs / (Selling Price per unit − Variable Cost per unit)
BEP (revenue) = Fixed Costs / Contribution Margin Ratio
Contribution Margin Ratio = (SP − VC) / SP
```

### Inventory Turnover
```text
Inventory Turnover = COGS / Average Inventory
Days Inventory Outstanding (DIO) = 365 / Inventory Turnover
```

### Demand during Lead Time
```text
Lead Time Demand = Average Daily Demand × Lead Time (days)
```

### ABC Analysis Thresholds (typical)
```text
A items: Top 10–20% volume → 70–80% of value  (tight control)
B items: Next 30%          → ~15% of value     (moderate control)
C items: Remaining 50%     → ~5% of value      (minimal control)
```
